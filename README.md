# ant.design

A ClojureScript library for [Ant Design](https://ant.design)


#### Quick Start Guide

Add the following project dependency:

```clojure
;; project.clj or build.boot
[ant.design "3.16.4-0"]

;; deps.edn
{:deps {ant.design {:mvn/version "3.16.4-0"}}}
```

# ant.design

#### Quick Start Guide

Add the following project dependency:

```clojure
;; project.clj
[ant.design "3.10.1-2-SNAPSHOT"]

;; deps.edn
{:deps {ant.design {:mvn/version "3.10.1-2-SNAPSHOT"}}}
```
